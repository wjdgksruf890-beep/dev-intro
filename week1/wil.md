오늘은 git hub 계정생성, git bash사용법 그리고 vercel로 배포하는 법까지 여러가지를 배우게 되었는데
일단 git bash의 명령어 git add, git commit, git push, git status등을 배우게되었다
개발을 아예 처음해보는 1학년 이기에 모든게 새로웠고 재밌었다. 앞으로도 개발에 대해서 더욱 재밌게 배울수 있는 계기가 될것같다 그리고 img 태그를 사용하여 이미지를 넣는법에 대해서도 공부를 하였는데 혼자서 개발에 대해서 공부도 해보니까 재밌었다. git add는 파일을 git bash에 넣는, git commit은 이름, git push 는 이것을 배포하는 명령어, git status는 지금 현재 파일의 상태를 보는 명령어 인것으로 배웠다(아마도..?) 첫날이라 배운것이 크게 얼마 있지는 않을것 같아 300자를 채우기는 힘들것 같습니다 ㅠ 아 글자수를 300자 이상 채워야지만 제출이 되는 군요. 일단 visual studio code를 이용해서 html파일을 작성합니다 그중 body 사이에 제목이나 파라그래프, 이미지 등 여러가지를 작성해준뒤에 git bash 를 이욯해서 git add, git status, git commit -m, git push등을 이용하여 배포를 해줍니다. 그뒤에 vercel에 들어가 배포가 완료되었는지 확인합니다. 그 중에 visual studio code에서 파일을 수정하였다면 git bash에 들어가 위와 같은 과정을 다시한번 해준뒤에 vercel에서 파일링크를 열어본뒤 위와 같은 과정이 완료되었는지 확인합니다.
